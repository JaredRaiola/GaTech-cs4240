# MIPS

How to run:

-cd to the directory

-TO MAKE:
  -run 'make'

-IF RUNNING NAIVE:
 -run 'make run filename="**YOUR TEST FILE HERE** -n"'

-IF RUNNING BRIGGS:
 -run 'make run filename="**YOUR TEST FILE HERE** -b"'

-TO CLEAN:
  -run 'make clean'

*NOTE*
Make sure to keep the quotes otherwise it will not work


*****ALL OUTPUT CAN BE FOUND IN THE 'Output.s' FILE*****
